## Usage

### Steam Game Information

To use the application, simply run it and enter a Steam AppID in the top-left input. Here are some examples!

- 252490 (Rust)
- 413150 (Stardew Valley)
- 504230 (Celeste)
- 730 (CSGO)
- 440 (TF2)
- 739630 (Phasmophobia)
- 1794680 (Vampire Survivors)
